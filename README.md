# RoguePulse_NewProject

一个完全独立的新项目（不依赖 `Week4_GameAI`），实现 3D Roguelike 动作射击核心玩法：

- 探索战斗 -> 经济成长 -> 传送器守点 -> 阶段推进 -> 胜负结算
- Build 路径：掉落 / 宝箱 / 商店 / 风险祭坛
- 升级三选一（`1/2/3`）
- 局外资源 Aether（`PlayerPrefs` 持久化）

## 快速启动

1. 用 Unity 打开 `RoguePulse_NewProject`
2. 执行菜单 `Tools/RoguePulse/Build New Run Scene`
3. 打开 `Assets/Scenes/Main.unity`
4. 点击 Play

## 操作

- `WASD`: 移动
- `Shift`: 冲刺
- `Space`: 跳跃
- `鼠标左键`: 射击
- `E`: 交互（宝箱/商店/祭坛/传送器）
- `1/2/3`: 选择升级
